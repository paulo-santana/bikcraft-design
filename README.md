## diário de bordo

### dia 1

Finalizados o wireframe e o design de estilo do site.  
Agora é partir pra codificação. Provavelmente primeiro o HTML e o CSS, depois um javascript pra consertar algumas  
coisas. Ou talvez o professor faça todo o conteúdo de uma página de cada vez, vamo ve...

### dia 2

Começamos e já terminamos a codificação da página index.html e sua devida estilização.  
Foi mais simples do que o que eu pensava que seria, mas por ser o primeiro projeto do curso, acho que está bom.  
O fato de especificar-mos todas as medidas em pixel é um pouco estranho, pois pelo que eu saiba, o ideal para web é deixar o layout bem adáptavel ao tamanho da tela (devido ao alto número de diferentes telas e resoluções no mercado).  
Vamos ver o que vai ser feito quando chegar as aulas de responsividade do curso.

### dia 3

Finalizei a página sobre e comecei a página produtos. Muita coisa já estava pronta por causa dos estilos padrão do arquivo style.css. Daí, foi só aproveitar as classes que já existiam e copiar alguns códigos delas delas pra novas classes. Por enquanto tá tudo em arquivos separados, mas depois tudo vai ser concatenado num arquivo só.

Depois de muito tempo eu aprendi uma coisa dahora do pseudo seletor nth-child: dá pra selecionar itens pares ou ímpares mandando ele fazer literalmente isso. Só colocar, no lugar de um número, as palavras **odd** ou **even**. A primeira pra itens ímpares, a segunda pra itens pares. Isso é bom pra diferenciar itens de uma tabela com muitas linhas ou com linhas muito longas, etc.
